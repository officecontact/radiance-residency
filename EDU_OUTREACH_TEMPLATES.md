# .edu / .ac.in Outreach Templates

> **Goal:** Get listed on college/university accommodation pages. Each .edu/.ac.in backlink passes high authority — far more valuable than typical directory backlinks.
>
> **Process:** Find each college's contact page (usually accommodation officer, student affairs, or webmaster). Send the relevant email. Follow up after 7 days if no reply.

---

## 📋 Email finder workflow

For each college:

1. Open their website
2. Look in the main navigation for: **"Accommodation"**, **"Hostels"**, **"International Students"**, **"Campus Life"**, or **"Off-Campus Living"**
3. If they have a list of nearby hostels — you want to be added there
4. Find the contact email — usually on a "Contact Us" page or via a `dean.student.affairs@...` / `accommodation@...` / `info@...` email
5. If no specific accommodation email, send to `info@` or `principal@`

If a college has a contact form instead of email, paste the same message into the form.
